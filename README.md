## **2. subcommands**
<a name="git-cleanup"></a>
### **git-cleanup**
_description:_ Delete local branches which does not exist on remote.

_usage:_  
```
git cleanup
```

<a name="git-cleanup"></a>
### **git-default-branch**
_description:_ Show default branch name.

_usage:_  
```
git default-branch
```
